Methodology

The data was sourced from an extensive study designed and conducted by
Columbia Business School. The data was converted to excel, cleaned of
nulls by giving replacement values, batched into cohort groups using
visual basic to create a new page per group, imported into pandas for
analysis and visualizations, and further cleaned. This final version of
the dataset is available on the "Data" page. The data was then analyzed
and rendered, then the elements were brought into html to create this website.
